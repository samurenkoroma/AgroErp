# AgroErp

**Статус проекта**: Начальная стадия разработки (Alpha)

## О проекте

AgroErp — это веб-приложение для планирования сельскохозяйственных работ. Основная идея — предоставить интерактивную карту хозяйства, где пользователь может:
- Выбирать объекты (поля, участки, теплицы).
- Планировать, какую культуру посеять/посадить.
- В будущем определять даты работ на основе погодных данных.

### Текущий функционал (Step 1)
- Отображение карты с объектами.
- При клике на объект открывается модальное окно.
- Для поля: выбор культуры для посева.
- Для участка/теплицы: отображение SVG-схемы с грядками и выбор культуры для каждой грядки.

> **Примечание**: Даты работ пока не учитываются, их планируется определять автоматически на основе погоды и других факторов.

## Технологический стек

- **Язык**: TypeScript (99.5%)
- **Фреймворк**: React (на основе Vite)
- **Стили**: Tailwind CSS
- **Сборка**: Vite
- **Дополнительно**: Используются компоненты из `temp.tsx` (вероятно, временный файл с прототипом)

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/samurenkoroma/AgroErp.git
   cd AgroErp
   npm install
   npm run dev
      ```
Откройте http://localhost:5173 в браузере.
