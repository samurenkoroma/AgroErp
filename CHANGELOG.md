## [0.0.3] - 2026-06-15

### Added

#### Модуль операций (Operations Module)
- **Страница операций (`OperationsPage`)** - управление задачами, операциями и таймлайном
    - Три вкладки: Задачи, Операции, Таймлайн
    - Статистика по задачам (всего, к выполнению, в работе, выполнено, просрочено)
    - Карточки задач с приоритетом и статусом
    - Карточки операций с иконками и payload данными
    - Таймлайн с группировкой по источникам (TASK, OPERATION, HARVEST, ALERT)
    - Модальные окна для создания задач и операций
    - Поиск и фильтрация по статусу

#### Модуль циклов выращивания (Growing Cycles Module)
- **Страница списка циклов (`GrowingCyclesPage`)** - отображение всех циклов выращивания
    - Таблица с колонками: Код, Название, Культура, Сорт, Статус, Плановый сбор
    - Статистика (всего, активных, сбор урожая, запланировано)
    - Фильтрация по статусу и поиск
    - Модальное окно создания цикла

- **Страница деталей цикла (`GrowingCycleDetailsPage`)** - полная информация о цикле
    - Четыре вкладки: General, Allocations, Plantings, Harvests
    - Отображение всей информации о цикле
    - Таблицы распределений, посадок и сборов
    - Модальные окна для добавления/редактирования

#### Модуль производственных объектов (Production Units Module)
- **Страница объектов (`ProductionUnitsPage`)** - управление полями, теплицами, контейнерами
    - Иерархическое отображение (дерево)
    - Четыре вкладки: Поля (с картой), Теплицы, Контейнеры, Гидропоника
    - Карта полей с отображением полигонов (Mapbox)
    - Древовидный список с раскрытием дочерних элементов
    - Детальная панель с информацией об объекте
    - Модальное окно создания объекта с динамическими полями размеров

- **Типы производственных объектов** - полная иерархия типов
    - Полевые: FIELD, BLOCK, BED, ROW
    - Тепличные: GREENHOUSE, GREENHOUSE_ZONE
    - Контейнерные: CONTAINER, RACK, SHELF, VERTICAL_TOWER
    - Посадочные места: SLOT, POT, TRAY
    - Гидропонные системы: NFT_CHANNEL, DWC_TANK, AEROPONIC_CHAMBER
    - Служебные: RESERVOIR, STORAGE
    - Гибридный тип PLOT (участок) с поддержкой всех возможностей

#### SCADA модуль (Временный, для демонстрации)
- **Страница SCADA мониторинга (`ScadaDashboard`)** - мониторинг датчиков в реальном времени
    - Группировка датчиков по объектам
    - Карточки с показателями температуры, влажности, CO2, освещенности
    - WebSocket для live данных
    - Алармы с цветовой индикацией
    - Мок-данные для демонстрации

#### Модуль инвентаря (Inventory Module)
- **Страница инвентаря (`InventoryPage`)** - управление материалами для рассады
    - Управление кассетами, стаканчиками и другими материалами
    - Статистика запасов
    - Фильтрация по типу и статусу
    - Прогресс-бары заполненности склада
    - История операций (приход/расход)

#### Плавающая кнопка действий (Floating Action Button)
- **Компонент `FloatingActionButton`** - быстрый доступ к действиям
    - Контекст для управления действиями на разных страницах
    - Меню с действиями (показывается при клике на главную кнопку)
    - Кнопка "Наверх" для прокрутки страницы
    - Автоматическое закрытие при клике вне области
    - Поддержка подтверждения опасных действий

#### Компоненты SVG схемы
- **`SvgSchemeEditor`** - редактор схемы грядок
    - Создание и редактирование грядок (добавление, удаление, изменение размера)
    - Два режима: редактирование и просмотр
    - Автоматический расчет площади грядок
    - Поддержка темной темы

#### Утилиты и хуки
- **`usePageActions`** - хук для установки действий плавающей кнопки на странице
- **`useCropIcons`** - утилита для получения иконок культур
- **`useStageIcons`** - утилита для получения иконок стадий развития
- **`statusLib`** - библиотека для работы со статусами
- **`dateLib`** - утилиты для форматирования дат
- **`geometry`** - утилиты для форматирования площади

#### Интеграция с API
- **Базовый API клиент** - поддержка JWT токенов
- **Refresh token механизм** - автоматическое обновление токенов
- **Auth API** - логин, регистрация, восстановление пароля
- **Organization API** - управление организациями/фермами
- **Мок-режим** - разработка без бэкенда с мок-данными

### Changed

- **Рефакторинг структуры проекта** - переход на feature-sliced архитектуру
- **Обновлена страница GreenhousePage** - вертикальные колонки, 2/3 для схемы, 1/3 для панели
- **Обновлена страница PlotPage** - использование новой структуры ProductionUnit
- **Улучшена страница FarmPage** - табы "Объекты" и "Посевы"
- **Обновлены страницы авторизации** - полная интеграция с JWT API
- **Улучшена страница дашборда** - добавлены виджеты активных посевов, задач, погоды
- **Обновлены стили** - полная поддержка темной темы во всех компонентах
- **Улучшен компонент Select** - добавлены поиск, иконки, группировка

### Removed

- **Удален устаревший SCADA модуль** - из production кода (оставлен только мок для демонстрации)
- **Удалены неиспользуемые компоненты** - старые версии модалок и таблиц
- **Удален устаревший SvgSchemeEditor** с избыточной функциональностью (зум, панорамирование)

### Fixed

- **Исправлена проблема с перезагрузкой страницы после логина** - оптимизирована гидратация auth store
- **Исправлена работа аккордеона в SvgSchemeEditor** - разделены обработчики кликов
- **Исправлен расчет площади в PlantingRecordModal** - учтено что площадь везде в гектарах
- **Исправлена ошибка "selectedChip is not defined"** в погодной полоске
- **Исправлен Warning о unique key** во всех списках компонентов
- **Исправлено переопределение действий плавающей кнопки** - через контекст
- **Исправлена проблема с раскрытием дочерних элементов** в дереве производственных объектов

## [0.0.2] - 2026-06-01

### Added

- **Страница управления сезонами (`SeasonsPage`)** - просмотр прошлых, текущего и планируемых сезонов
    - Левая колонка с группировкой сезонов по типам
    - Правая колонка с детальной информацией о выбранном сезоне
    - Статистика по сезону (площадь, урожай, эффективность)
    - Выпадающий блок с грядками/участками под каждой культурой
    - Модальные окна для создания/редактирования сезона

- **Страница всех посевов (`AllCropsPage`)** - полный список планов посева
    - Два режима просмотра: сетка и список
    - Статистика (активные, завершенные, запланированные)
    - Поиск и фильтрация по статусу и типу объекта
    - Карточки планов с прогресс-баром
    - Модальные окна для создания/редактирования/удаления

- **Страница каталога культур (`CropPage`)** - управление справочником культур
    - Сетка культур с иконками и категориями
    - Поиск и фильтрация по категории
    - Детальная страница культуры с сортами
    - Детальная страница сорта с характеристиками

- **Виджет посевов (`CropsWidget`)** - отображение на главной странице фермы
    - Статистика (активные, завершенные, запланированные)
    - Поиск и фильтрация по статусу
    - Прогресс-бары для каждого посева
    - Индикация рассадного метода

- **Модалка создания посева (`CreateCropPlanModal`)** - с поддержкой прямого посева и рассадного метода
    - Выбор культуры, сорта, места посадки
    - Настройка параметров рассады (возраст, высота, листья, закалка)
    - Автоматический расчет даты посева семян
    - Автогенерация названия

- **Модалка фиксации посадки (`PlantingRecordModal`)** - для заполнения истории
    - Выбор культуры, сорта, сезона
    - Расчет необходимого количества семян/рассады
    - Фиксация дат посадки и сбора
    - Учет фактической урожайности

- **Компонент FloatingActionButton** - для быстрых действий
    - Плавающая кнопка с меню действий
    - Кнопка прокрутки наверх

### Changed

- **Обновлен дизайн страницы GreenhousePage** - схема слева, панель справа
- **Улучшена навигация** - добавлены хлебные крошки
- **Оптимизированы запросы API** - добавлены интерсепторы для работы с JWT

### Fixed

- **Исправлена ошибка с refresh token** - корректное обновление токенов при 401 ошибке
- **Исправлено отображение модалок** - правильное закрытие по Escape и клику вне
- **Исправлены TypeScript ошибки** - во всех новых компонентах

## [0.0.1] - 2026-05-15

### Added

- **Начальная настройка проекта**
    - Vite + React + TypeScript
    - Tailwind CSS для стилизации
    - React Router для маршрутизации
    - TanStack Query для работы с API
    - Zustand для управления состоянием
    - Next Themes для поддержки темной/светлой темы
    - Lucide React для иконок
    - Mapbox GL для карт

- **Авторизация и аутентификация**
    - Страницы входа и регистрации
    - Хранение JWT токенов
    - Приватные маршруты
    - Защита от неавторизованного доступа

- **Базовый Layout**
    - Боковое меню с навигацией
    - Шапка с профилем пользователя
    - Поддержка сворачивания бокового меню
    - Переключение темы (светлая/темная)

- **Страница дашборда**
    - Статистика по ферме
    - Активные посевы с прогрессом
    - Прогноз погоды на 5 дней
    - Список активных задач
    - Предупреждение о низком запасе в инвентаре
    - Быстрые действия

- **Страница фермы (`FarmPage`)**
    - Карта с полями, теплицами, участками
    - Панель со списком объектов
    - Табы для переключения между объектами и посевами

- **Страница теплицы (`GreenhousePage`)**
    - Отображение SVG схемы теплицы
    - Редактирование схемы (добавление/удаление грядок)
    - Информационные табы (информация, климат, культуры, задачи)
    - Возможность планирования посадки при клике на грядку

- **Страница участка (`PlotPage`)**
    - Аналогична странице теплицы
    - Поддержка SVG схемы с грядками

- **Базовый SVG редактор (`SvgSchemeEditor`)**
    - Простое редактирование схемы грядок
    - Добавление, удаление, изменение размера грядок
    - Перетаскивание грядок мышью
    - Два режима: редактирование и просмотр

- **API клиент**
    - Базовые методы GET, POST, PUT, DELETE
    - Интерсепторы для добавления токена
    - Обработка ошибок

- **Мок-данные**
    - Для разработки без бэкенда
    - Модели культур, сортов, планов посева

### Added (Mobile)

- Адаптивная верстка для мобильных устройств
- Улучшенная навигация на маленьких экранах

### Added (Infrastructure)

- ESLint и Prettier для форматирования кода
- Husky для pre-commit хуков
- GitHub Actions для CI/CD
- Dockerfile для контейнеризации

### Changed (Initial)

- Базовая структура проекта
- Настройка окружения (development/production)
